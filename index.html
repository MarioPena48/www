Ruta de Instalación

  • Agendamos el levantamiento

    Programamos una cita contigo para saber las necesidades y así mismo poder realizar una propuesta de valor.

  • Visita del equipo EDOTEC para revisión en campo 

    Revisamos la instalación eléctrica para determinar lo que necesitamos hacer para sacar el máximo de provecho a nuestra instalación domótica, también verificamos la red wifi para asegurar que los dispositivos tengan el alcance  para su perfecta señal. La instalación es mínima, en caso de apagadores solamente verificar que tengamos un neutro. Toda la demás instalación no es invasiva. 

  • Realización de propuesta domótica y cotización

    Teniendo el levantamiento trabajamos en un proyecto de valor basado en las necesidades. Cotización de dispositivo y de instalación de nuestros equipos

  • Instalación según el proyecto y programación del sistema

    Una vez autorizado el proyecto y los dispositivos listos para su instalación, vamos al sitio para instalar y programar todos los equipos así como dispositivos proyectados. 
    Capacitación de uso del sistema y programación de escenas y automatizaciones.
